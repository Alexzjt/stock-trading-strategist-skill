# 股市趋势技术分析：经典图表形态与交易战术

> 来源：约翰·迈吉、罗伯特·爱德华兹《股市趋势技术分析》（原书第10版）
> 适用场景：审查中长线趋势、经典反转形态（头肩/双头/三角形）、突破真实性量化验证、测算目标价位。

---

## 一、 真假突破的量化过滤器法则

散户最容易犯的致命错误是：价格稍微越过阻力位，或盘中瞬间突破，就激动追高买入（常死于“诱多”或“多头陷阱”）。必须强制执行以下量化过滤：

### 1. 突破空间过滤器：3% 法则
- **核心定义**：决定性的突破，要求收盘价必须超过阻力位/支撑位水平至少 **3%**。
- **应用场景**：如果阻力位在 100 元，股价必须收盘在 103 元以上，才算有效突破。101 元、102 元的突破极大概率是假突破。

### 2. 突破时间过滤器：3日法则 / 周末法则
- **核心定义**：价格突破后，必须在阻力/支撑位之外**连续收盘站稳至少 3 个交易日**，或者在**周五收盘时**站稳该位置。

### 3. 成交量的验证定律（最核心）
*注：迈吉在原著中并未给出如“放量150%”这类现代死板公式，而是通过相对的“显著放大”和“枯竭”来定性。*
- **向上突破的标准（显著放量）**：**向上突破必须伴随成交量的显著放大！** 这里的“放大”是相对前期盘整或形态构筑期间的萎缩而言。如果突破时交投清淡，缺乏明显的买盘爆发，则不能确认突破（极有可能是诱多假突破），绝不能买入。
- **警惕“抛售高峰/单日反转”（Climax Volume）**：出现在长期涨势末端或极度恐慌下跌末端的天量（原著中甚至出现过“两倍于之前牛市顶部任何一天”的极端成交量）。在上涨末端，这种异常的放量冲高往往意味着**主力派发与趋势终结（清仓日）**，而非健康突破的开始。
- **向下破位的标准（无需放量）**：**向下突破不需要成交量配合！** 华尔街经典法则：“股票涨起来需要买盘推动，跌起来就像自由落体。”无量跌破支撑线，形态依然成立，而且往往跌得更深。绝不能因“无量下跌”就抱有幻想而死扛。

**👉 Agent 审查口径**：
当用户说“股票放量突破了阻力，准备买入”时，必须三连问：
1. **幅度达标了吗？** 突破幅度超过 3% 了吗？
2. **时间达标了吗？** 站稳 3 天了吗？
3. **量能相对显著吗？** 相比前期的缩量盘整，此时成交量是否出现了肉眼可见的“显著放大”？（如果是无量突破则否决；如果是极端的“天量滞涨”，警告这可能是抛售高峰，切勿盲目追高）。

## 二、 经典反转形态的确立纪律

### 1. 头肩顶（Head and Shoulders Top）
头肩顶是所有反转形态中最可靠、最经典的一个。
- **阶段一（左肩）**：伴随巨大成交量的强劲上涨，随后伴随成交量萎缩的回调。
- **阶段二（头部）**：再次上涨，创出新高（高于左肩），但**成交量明显低于左肩**（这是极其危险的见顶警报）。
- **阶段三（右肩）**：价格第三次上涨，但高点低于头部，且**成交量非常萎缩**。
- **阶段四（确认突破）**：价格跌破连接两个回调低点的“颈线”（Neckline），收盘跌幅超过 3%。此时无需大成交量配合，形态即告成立。

**👉 Agent 审查口径**：如果用户想在头肩顶的右肩买入，警告他“成交量正在枯竭，反弹无力，随时面临颈线破位”。如果跌破颈线，强制要求清仓止损。

### 2. 双重顶/底（M头 / W底）
- **时间跨度要求**：两个顶点/底点之间，至少需要相隔 **1个月** 以上的交易时间。那些几天内走出的所谓“双头”，大多只是普通的整理洗盘。
- **量价特征**：双重顶的第二个高点，成交量必须**显著小于**第一个高点。双重底的第二个低点，往往成交量极度萎缩，但随后的向上突破颈线，必须**放出巨量**。
- **成立条件**：没有跌破（或涨破）两峰（或两谷）之间的颈线前，形态不成立！不要提前预判“现在是双底可以买了”。

### 3. 对称三角形（Symmetrical Triangles）
- **突破时机**：价格通常在三角形从左到右 **1/2 到 3/4** 的距离处发生突破。如果价格一直走到三角形的尖端（顶点），这种突破往往毫无力度，会变成横盘死水。
- **成交量**：三角形构建期间，成交量必须**持续萎缩**。向上突破时，必须伴随大成交量。

## 三、 支撑与阻力的角色互换与确认

- **角色互换**：一旦一个显著的阻力位被有效突破（符合 3% 和成交量法则），它就立刻变成了未来的**强支撑位**；反之亦然。
- **回抽测试（Throwback / Pullback）**：价格在突破后，通常会在几天内缩量回调，重新测试该支撑/阻力线。
  - **健康的回抽**：缩量，且绝不能重新跌破（或涨回）颈线。一旦跌破，说明之前的突破是“假突破”。
  - **交易战术**：回抽测试结束，价格再次顺着突破方向启动时，是绝佳的安全买点/加仓点。

## 四、 测算目标位（垂直投影法）

不要让用户凭空想象“我觉得能涨到多少”。引入科学的最低测算目标：

### 1. 头肩形态测算
- **测量方法**：计算从**头部极点（最高/最低价）** 垂直向下（或向上）到**颈线**的距离（即垂直高度 $H$）。
- **投影目标**：将距离 $H$ 从**颈线被突破的点** 向下（或向上）垂直投影，该位置就是第一测算目标位。

### 2. 三角形与矩形测算
- **测量方法**：取形态开始形成时的**最宽处垂直距离**。
- **投影目标**：从突破点顺势投影该距离。

**👉 Agent 审查口径**：当用户设定止盈位时，询问其目标价是如何得出的？要求其使用形态的垂直投影法进行客观测算，拒绝毫无根据的“翻倍”幻想。

## 五、 约翰·迈吉的“基准点”法（Basing Points）
*这是替代道氏理论判断中长线趋势反转的利器。*

- **多头基准点**：在上升趋势中，最近一次有效回调的**最低点**。只要价格不跌破这个基准点，长线多头趋势就未改变，任何震荡都是洗盘。
- **空头基准点**：在下降趋势中，最近一次反弹的**最高点**。只要反弹没有越过该基准点，长线空头趋势就未改变，任何上涨都是“死猫跳”（逃命波）。

**👉 Agent 审查口径**：如果用户因短期下跌感到恐慌想卖出，询问他：“**价格跌破前一个多头基准点了吗？** 如果没有，趋势未变，按兵不动。”
