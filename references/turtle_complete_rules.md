# 海龟完整交易系统规则（附录）

> 来源：Curtis Faith《海龟交易法则》
> 适用场景：系统化交易决策、开仓/加仓/止损/退出的完整决策流程

---

## 一、完整交易系统六要素

一个完整的交易系统必须涵盖所有决策，**不给主观判断留余地**：

| 要素 | 核心问题 | 海龟的答案 |
|------|---------|-----------|
| **市场** | 买卖什么？ | 高流动性期货（A股适配：主板蓝筹/主流ETF） |
| **头寸规模** | 买卖多少？ | N头寸单位法（ATR标准化） |
| **入市** | 什么时候买卖？ | 20日/55日突破 |
| **止损** | 什么时候放弃亏损头寸？ | 2N止损，无一例外 |
| **退出** | 什么时候退出盈利头寸？ | 10日/20日反向突破 |
| **战术** | 怎么买卖？ | 限价单优先，急变市场保持冷静 |

## 二、入市系统

海龟使用两套入市系统，可单独或组合使用：

### 系统1：20日突破（短期）
```
入市信号：价格突破过去20日最高点（做多）
          价格跌破过去20日最低点（做空）

过滤规则：若上一次突破是盈利性突破，本次信号忽略
          → 转而等待55日突破点入市（保障性触发）

亏损性突破定义：突破后价格在退出前发生了2N的不利变动
```

### 系统2：55日突破（长期）
```
入市信号：价格突破过去55日最高点（做多）
          价格跌破过去55日最低点（做空）

过滤规则：无过滤，所有突破都是有效信号
```

> **统一性原则**：一年的大部分利润可能来自2-3次大趋势，错过一次信号可能大幅降低年化收益。

## 三、逐步建仓规则

```
步骤1：突破信号触发，买入第1个头寸单位
步骤2：价格上涨 1/2N，买入第2单位
步骤3：价格再涨 1/2N，买入第3单位
步骤4：价格再涨 1/2N，买入第4单位（满仓上限）
```

**黄金示例**：
```
N = 2.50，55日突破价 = 310.00

第1单位：310.00
第2单位：311.25（+1.25）
第3单位：312.50（+1.25）
第4单位：313.75（+1.25）
```

## 四、止损规则

### 标准止损：2N止损
```
多头止损 = 入市价 - 2N
空头止损 = 入市价 + 2N
```

### 多单位组合止损调整

| 建仓阶段 | 第1单位入市价 | 第2单位入市价 | 第3单位入市价 | 第4单位入市价 | 统一止损价 |
|---------|-------------|-------------|-------------|-------------|---------|
| 第1单位 | 28.30 | — | — | — | 25.90 |
| 第2单位 | 28.30 | 28.90 | — | — | 26.50 |
| 第3单位 | 28.30 | 28.90 | 29.50 | — | 27.10 |
| 第4单位 | 28.30 | 28.90 | 29.50 | 30.10 | 27.70 |

> 每新增一个单位，所有已持单位的止损价**同步上移1/2N**。

## 五、退出规则（盈利头寸退出）

> "趋势跟踪者最常犯的错误就是过早退出盈利头寸。"

### 系统1退出：10日突破退出
```
多头：价格跌破过去10日最低点 → 全部退出
空头：价格突破过去10日最高点 → 全部退出
```

### 系统2退出：20日突破退出
```
多头：价格跌破过去20日最低点 → 全部退出
空头：价格突破过去20日最高点 → 全部退出
```

> **退出纪律**：必须等到反向突破信号出现才退出。允许账户利润在退出前回撤20%~40%甚至更多，这是趋势跟踪的必然代价。

## 六、四层头寸上限（风险控制）

| 层面 | 范围 | 最大单位数 |
|------|------|----------|
| 1 | 单个市场/个股 | 4 |
| 2 | 高度关联市场（同方向） | 6 |
| 3 | 松散关联市场（同方向） | 10 |
| 4 | 任意单方向总计 | 12 |

**高度关联示例**：原油+民用燃料油、黄金+白银、所有外汇
**松散关联示例**：黄金+铜、白银+铜

## 七、战术执行细节

### 下单原则
- **优先使用限价单**，而非市价单
- 限价单成交价通常更有利，滑点更小
- 不向经纪人提前透露止损单位置（防止被猎杀）

### 急变市场处理
```
错误做法：惊慌失措，立即下市价单 → 往往在最差点位成交

正确做法：保持冷静，等市场稳定后再行动
          若稳定后价格已超过止损位，平静退出
```

### 买强卖弱原则
同时出现多个信号时：
- **做多**选择最强势的市场/个股（突破幅度最大、相对N值最高）
- **做空**选择最弱势的市场/个股

**强弱衡量方法**：
```
标准化强度 = (当前价格 - 3个月前价格) / N

结果最高 → 最强（适合做多）
结果最低 → 最弱（适合做空）
```

## 八、账户规模管理

```
初始账户 = 100万元

触发条件：账户净值下跌10%（亏损10万元）
操作：名义账户规模缩减20% → 调整为80万元
     继续亏损10万元（即80万的10%=8万）
     再次缩减20% → 调整为64万元
```

> 账户缩减是为了**降低亏损期的交易规模**，保护剩余资本。恢复到年初净值后，方可恢复原始规模。

## 九、心理与纪律

### 四大成功要素
> 海龟们的成功心得，按重要性排序：
1. **掌握优势**：找到长期期望值为正的策略
2. **管理风险**：控制风险，守住阵地，否则等不到系统发挥作用
3. **坚定不移**：唯有坚持执行，才能真正获得系统的正期望值
4. **简单明了**：长久来看，简单的系统比复杂的系统更有生命力

### 理查德·丹尼斯的告诫
> "我说过很多次，你可以把我的交易法则登在报纸上，但没人会遵守它们。关键是统一性和纪律性。几乎每一个人都可以列出一串法则，而且不比我们的那些法则差多少。但他们不能给别人信心，而唯有对法则充满信心，你才会坚持这些法则，即使遭遇逆境。"

### 常见失败模式
| 失败模式 | 具体表现 |
|---------|---------|
| 怀疑法则 | 认为"一定还有秘诀没透露" |
| 修改法则 | 微小调整（如加仓节奏放慢）导致止损频繁触发 |
| 过早退出 | 看到利润回撤就急于锁定，错过大趋势 |
| 风险过大 | 仓位是合理水平的5-10倍 |
| 追求过多 | 以金钱为目标导致心理失衡，执行力崩溃 |

## 十、A股适配备注

海龟系统源自期货市场，A股应用时注意：
1. **无做空机制**：系统1/2的空头信号在A股不可用，仅应用多头部分
2. **T+1限制**：当天买入无法当天卖出，急变市场退出有滑点
3. **市场选择**：优先选择流动性充足的主板蓝筹、主流ETF，避免小市值股
4. **头寸单位换算**：以"手"（100股）为最小单位，向下取整
5. **N值更新**：建议每周更新一次ATR，而非实时计算
