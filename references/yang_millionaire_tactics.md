# 杨百万（杨怀定）A股量化与趋势战法

本文档提炼自《做个百万富翁——杨百万自述》，旨在为股票导师 Agent 提供基于中国A股早期历史经验沉淀下来的**宏观周期判断标准**与**技术指标抄底逃顶**的量化信号。

## 核心法则：大势识别与股市循环的十个阶段

导师 Agent 必须具备通过宏观周期判断个股系统性风险的能力。杨百万将股市循环划分为十个阶段，导师在进行技术审查时，必须首先判定当前大盘处于哪个阶段：

1. **低迷阶段**：坏消息满天飞，悲观情绪蔓延，成交量极度萎缩（价跌量缩）。此时是**大底部**，具有实力的投资人开始悄悄进货。
2. **发动阶段**：低价股/绩差股开始活跃领涨，成交量不规则递增。
3. **反动（回档）阶段**：机构打压洗盘，价跌量缩，但不创新低。这是中小散户长线建仓的最佳时机。
4. **主升阶段**：价升量增，K线拉出长阳，甚至全面暴涨。新股民开始疯狂入市。**（此时机构开始伺机出货）**
5. **盘整阶段**：由于股价炒高，出现获利回吐，题材满天飞。
6. **末升阶段**：所谓“黑马”频出，绩优股反而滞涨，成交量创出历史天量。指数攀升困难，高位频频出现十字星。**（极度危险，随时崩盘）**
7. **初跌阶段**：少数垃圾股拉长阳诱多，机构大举抛售，成交量相对于末升段有所萎缩。**（必须果断清仓离场）**
8. **主跌阶段**：越跌越惨，价跌量增（恐慌性抛盘），新股跌破发行价。
9. **反弹阶段（逃命期）**：暴跌后短暂回稳，上涨无量，下跌放量。**（严厉警告散户：绝不可在此阶段抢反弹补仓，应趁机逃命）**
10. **末跌阶段**：股价跌幅巨大，重回低迷阶段，机构开始重新分批吸纳。

## 抄底逃顶的五大核心信号体系

在日常研判中，导师 Agent 需要结合以下五个维度的共振来给出操作建议，严禁依赖单一指标：

### 1. 移动平均线 (MA) 的拐点与趋势
* **基本纪律**：股价从下方突破均线并站稳，均线向上拐头，具有助涨特性；股价跌破均线，均线向下拐头，形成死叉，具有助跌特性。
* **高低位判定**：坚决摒弃盲目操作，当 5日均线与 10日均线在高位发生“死亡交叉”时，必须强制止损出局。

### 2. K线形态的极端心理反映
* **顶部形态 (高危区)**：在长期上升或主升浪后，出现**“乌云盖顶”**（高开走低的大阴线，伴随剧烈放量），或者高位出现**“十字星”**、**“上吊线”**，这是极其危险的看跌信号。
* **底部形态 (建仓区)**：在长期下跌末期，出现**“看涨吞没”**或带长下影线的**“锤子线”**。

### 3. 成交量 (Volume) 的“天量天价”定律
* **量价背离定律**：股票上涨必须有量配合。如果高位出现“上涨无量，下跌放量”的量价背离，是典型的出货行情。
* **见顶信号**：经过长期上升后，成交量连续维持在天量级别，但股价却无法继续上涨（风起云涌，股价不涨），一旦成交量开始萎缩，股价必定暴跌。

### 4. MACD 的顺势与背离战法
* **牛熊背离**：
    * **顶背离 (牛背离)**：股价创出新高，但 MACD 红柱不再创新高，甚至出现死叉，暗示将要暴跌。
    * **底背离 (熊背离)**：股价创出新低，但 MACD 绿柱没有创新低，暗示即将见底反弹。
* **实战口诀**：强势市场看均线交叉，弱势市场看柱状体缩短。在特强市道中，MACD 快慢线两次向下交叉，**必须清仓**。

### 5. RSI (相对强弱指标)
* 导师 Agent 应结合**周线级别的 RSI 背离**来作为极高准确率的抄底逃顶长线指标。周 RSI 高位钝化并发生顶背离，往往是大级别熊市的开端。

## 导师 Agent 核心教条附加

> “做股票绝不能指望一夜暴富，买卖股票实际上是在买卖心态。在股市极热的时候要克制贪婪，在股市低潮时要战胜恐惧。对于处于长期下降通道（主跌段）的股票，即使它有短暂的反弹，也绝对不允许散户盲目去接飞刀！”
