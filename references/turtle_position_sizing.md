# 海龟交易法则：N头寸规模法与风险管理

> 来源：Curtis Faith《海龟交易法则》
> 适用场景：仓位计算、止损设定、资金管理审查

---

## 一、N（ATR）的定义与计算

**N** 即真实波动幅度均值（ATR），代表市场在一天内的**平均波动幅度**，包含跳空缺口。

### 每日真实波动幅度公式
```
真实波动幅度 = Max(H-L, H-PDC, PDC-L)

H   = 当日最高价
L   = 当日最低价
PDC = 前一日收盘价
```

### N值（ATR）的计算公式
```
N = (19 × PDN + TR) / 20

PDN = 前一日N值
TR  = 当日真实波动幅度
```
> 首次计算 N 时，使用过去20日真实波动幅度的简单平均值。

---

## 二、波幅调整头寸单位规模

海龟系统的核心原则：**让1个ATR的价格变动恰好等于账户净值的1%**。

### 头寸单位规模公式
```
头寸单位规模 = (账户净值 × 1%) / (N × 每一点所代表的美元)
```

### 计算示例（A股适配版）
```
假设：
  账户净值  = 500,000元
  某股票N值 = 1.20元（20日ATR）
  风险比例  = 1%

头寸单位规模 = (500,000 × 1%) / 1.20
             = 5,000 / 1.20
             ≈ 4,166 股（约4手，取整）
```

> **关键原则**：不同市场/股票的头寸大小，由其波动性决定，而不是由价格决定。
> 波动性大的股票，买得少；波动性小的股票，买得多。

## 三、四层风险控制上限

海龟系统对总体头寸规模设有**4个层面**的限制，防止集中风险：

| 层面 | 限制范围 | 头寸单位上限 |
|------|----------|-------------|
| 1 | 单个市场/个股 | 4个单位 |
| 2 | 高度关联的多个市场 | 6个单位（同方向） |
| 3 | 松散关联的多个市场 | 10个单位（同方向） |
| 4 | 单个方向（多头或空头）合计 | 12个单位 |

**A股实操说明**：
- 同一只股票最多持有4个单位仓位
- 同板块（如银行股）多只合计不超过6个单位
- 同方向总持仓不超过12个单位

## 四、逐步建仓法则（金字塔加仓）

海龟不在突破时满仓买入，而是**分批建仓**：

```
突破点建立第1单位
涨 1/2N 后，建立第2单位
再涨 1/2N，建立第3单位
再涨 1/2N，建立第4单位（达到上限）
```

### 原油示例
```
N = 1.20，55日突破价 = 28.30

第1单位：28.30
第2单位：28.30 + 0.60 = 28.90
第3单位：28.90 + 0.60 = 29.50
第4单位：29.50 + 0.60 = 30.10
```

**A股加仓逻辑**：
```
N = 1.50元，突破价 = 20.00元

第1单位：20.00元
第2单位：20.75元（+0.75，即1/2N）
第3单位：21.50元
第4单位：22.25元
```

## 五、止损法则

### 标准止损规则（2N止损）
```
单笔交易最大风险 = 账户净值的2%
止损点 = 入市价 - 2N（多头）
止损点 = 入市价 + 2N（空头）
```

### 多单位建仓时的止损调整
每新增一个单位，前面所有单位的止损点**上移1/2N**，保持整体仓位的止损价一致：

```
原油入市过程（N=1.20，突破=28.30）：

建立第1单位后：    入市28.30，止损25.90（-2N）
建立第2单位后：    第1单位止损升至26.50
                  第2单位止损26.50（入市28.90-2N）
建立第3单位后：    1、2、3单位止损统一为27.10
建立第4单位后：    1、2、3、4单位止损统一为27.70
```

> 核心：整个头寸组合的止损价始终一致，不会因先买入的单位持续承压。

### 备选止损策略：双重损失法（Whipsaw）
- 每单位止损压缩至 **1/2N**（账户风险0.5%）
- 止损触发后，等价格回到**原入市价**再重新建仓
- 优点：胜率下降，但利润率更高；适合有纪律的交易者

## 六、风险量化核心公式汇总

| 指标 | 公式 | 含义 |
|------|------|------|
| N（ATR） | (19×PDN + TR) / 20 | 市场日均波动幅度 |
| 头寸单位 | (账户净值×1%) / (N×点值) | 单次建仓数量 |
| 止损价（多） | 入市价 - 2N | 标准止损位 |
| 最大风险/笔 | 账户净值×2% | 单笔最大损失 |
| 加仓间距 | 1/2N | 每次追加仓位的间距 |

## 七、资金管理核心原则

### 生存第一
> "交易的首要目标应该是生存。时间站在你这边。一个期望值为正的系统早晚会给你带来财富，但这一切有一个前提——你必须留在游戏场中。"

### 破产风险的主要来源（非随机，而是人为错误）
1. **没有计划**：依赖直觉、传闻、猜测行动
2. **风险过大**：仓位是合理水平的5-10倍
3. **期望不切实际**：相信自己能快速赚大钱

### 衰落控制准则
- 历史模拟中，衰落幅度最大不超过**可承受极限的1/2**
- 账户亏损10%时，主动将名义账户规模**缩减20%**
- 每年20%-30%的稳定复利远优于追求100%以上的激进回报

### 为何波动性标准化如此重要
> "有效的资金管理能帮助你安然度过对每一个交易者来说都不可避免的逆境时期。"
> 
> 不根据波动性调整头寸的简单方法（如"固定手数"），会导致某些市场权重过高，破坏分散化效果。

## 八、常见错误警示

| 错误行为 | 为何危险 |
|----------|---------|
| 不设止损 | 一次大行情可能清空账户（1987年案例） |
| 固定手数不调整 | 低波动性仓位过重/高波动性仓位过轻 |
| 越跌越买（补仓） | 无法量化风险，打破资金管理约束 |
| 超出头寸单位上限 | 失去分散化保护，遭遇集中性冲击时损失倍增 |
| 过早止盈 | 趋势跟踪系统盈利来自少数大趋势，截断利润等于截断系统盈利能力 |
